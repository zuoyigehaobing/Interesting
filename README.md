###

    navigate to Downloads: dow

    navigate to Documents: doc

    打开portal：portal

    打开acorn：acorn

    打开github：github

    创建文件夹并进入：mcd + 文件名

    压缩文件夹： zipf + 文件名

    在Documents中寻找文件： where + 文件名
